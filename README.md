### 💻프로젝트 개요

- 시중에 일정관리하는 웹캘린더는 많지만, 공부하는 사람들을 위한 스터디플래너는 마땅치 않은 상황입니다. 모바일에서 사용가능한 스터디플래너 앱은 많지만, 주변에서 공부 중 핸드폰을 사용하니 집중력이 흐트러진다는 불편함을 호소하는 모습을 볼 수 있었습니다. 우리 팀은 이러한 불편함을 개선하고자, 웹을 통해 사용할 수 있는 스터디플래너를 구상하게 되었습니다.
